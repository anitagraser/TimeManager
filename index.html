    • QGIS Time Manager Plugin

     

    Project maintained by anitagraser Hosted on GitHub Pages — Theme by mattgraham

    The aim of "Time Manager plugin for QGIS" is to provide comfortable browsing through temporal geodata. A dock widget provides a time slider and a configuration dialog for your layers to manage.

    This is an example video I created using Time Manager.

    What Time Manager does

    Time Manager filters your datasets and displays only features with timestamps in the user specified time frame. The plugin package contains help documents and test data to get accustomed to Time Manager.

    More information on functionality and limitations can be found on Github.

    Where to download Time Manager

    Time Manager is available through QGIS Plugin Repository http://plugins.qgis.org/plugins/timemanager/
